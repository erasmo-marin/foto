Foto is an Image Viewer and Album Manager for the elementary desktop written in Vala and Gtk 3

Installing (Ubuntu 14.04 - 14.10, Elementary Freya or equivalent)

$ sudo add-apt-repository ppa:erasmo-marin/foto-1.0
$ sudo apt-get update
$ sudo apt-get install foto

Building from source (check the INSTALL file for more details)

$ bzr branch lp:foto/foto-1.0
$ cd foto-1.0
$ mkdir build
$ cd build
$ cmake .. -DCMAKE_INSTALL_PREFIX=/usr
$ make
$ sudo make install
